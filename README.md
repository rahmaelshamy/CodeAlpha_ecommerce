# CodeAlpha E-commerce Store

A mini full-stack e-commerce web application built as part of the CodeAlpha Full Stack Development Internship.

## Features
- User registration and login (JWT authentication)
- View product listings
- Product details page
- Add products to cart
- View cart
- Place orders
- Logout functionality

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- SQLite
- JWT Authentication

## How to Run
1. npm install
2. node server.js
3. Open http://localhost:5001/login.html
